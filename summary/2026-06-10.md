## 주요 이슈 TOP 3
1. **Anthropic, 글로벌 AI 개발 일시정지 촉구 — "Claude가 전체 코드의 80% 생산"** — Claude의 오픈엔드 코딩 성공률이 6개월 만에 50%p 상승해 76%에 도달했으며, 엔지니어 1인당 코드 병합량은 2024년 대비 8배에 달한다. Anthropic은 이를 "재귀적 자기개선" 임박 신호로 보고, 검증 가능한 국제 체계 아래 프론티어 AI 개발을 일시 정지해야 한다고 주장했다. 단, 자사만의 일방적 중단이 아닌 타 주요 연구소의 동참을 조건으로 제시했다. — [출처](https://www.sofx.com/anthropic-calls-for-global-ai-pause-as-claude-writes-its-own-code/)

2. **Microsoft Build 2026: OpenAI 의존 없는 자체 AI 모델 7종 발표** — MAI-Thinking-1(35B 파라미터, AIME 2025 97.0% 달성)과 MAI-Code-1-Flash(5B, SWE-Bench Pro 51.2%)를 포함한 7개 모델을 발표했다. MAI-Code-1-Flash는 즉시 GitHub Copilot 전 플랜에 배포되며, GitHub Copilot은 같은 날 요청 기반에서 사용량 기반 과금으로 전환을 선언했다. Microsoft 역사상 OpenAI 데이터 없이 훈련된 최초의 인하우스 추론 모델이라는 점에서 AI 공급망 다변화의 신호탄으로 평가된다. — [출처](https://www.cnbc.com/2026/06/02/microsoft-unveils-new-ai-models-lessen-reliance-on-openai-lower-costs.html)

3. **WWDC 2026: Apple Siri, 구글 Gemini 1.2조 파라미터 모델로 전면 교체** — Apple은 연간 약 10억 달러에 Google과 계약을 맺고, Siri를 ChatGPT·Claude와 유사한 대화형 인터페이스로 재설계했다. 신형 Siri는 시스템 전역 'Search or Ask' 제스처와 Dynamic Island 통합을 포함하며, Apple Foundation Models 전체가 Gemini 패밀리와 협력 개발된 것으로 확인됐다. — [출처](https://techcrunch.com/2026/06/09/wwdc-2026-everything-announced-on-siri-ai-os-27-apple-intelligence-and-more/)
