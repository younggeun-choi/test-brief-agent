## 주제

- 생성형 AI 동향

## 작업 순서

1. 한국 시간 기준 오늘 날짜를 확인하라.
2. WebSearch와 WebFetch로 최근 24시간 이내의 관련 자료를 수집하라.
3. 한국어/영어 자료를 모두 포함하되, 신뢰도 낮은 SNS·개인 블로그는 제외하라.
4. 항목 1개당 다음 정보를 정리하라:
   - 제목
   - 출처 URL
   - 발행 시각 (가능한 경우)
   - 본문 핵심 3~5줄 (원문 그대로 인용, 요약·해석 금지)
5. 최소 5개, 최대 15개 항목을 수집하라.
6. 결과를 `raw/YYYY-MM-DD.md` 경로로 새 파일을 생성하고 커밋·푸시하라.
   - YYYY-MM-DD는 오늘의 한국 날짜
   - 커밋 메시지: `scrap: YYYY-MM-DD`

## 출력 파일 형식

```
# Raw — YYYY-MM-DD

## 1. (제목)
- 출처: (URL)
- 발행: (시각)
- 본문:
  (원문 핵심 3~5줄)

## 2. ...
```

## 금지 사항

- 24시간 이전 정보 포함 금지
- 본문 요약·해석·의견 추가 금지 (요약은 briefing 단계에서 수행)
- 같은 출처를 중복으로 수집 금지
- 저작권상 문제될 수 있는 전문 전체 인용 금지 (핵심 부분만)
